High order functions

Useful?

  • Domain specific languages can be defined as collections of higher-order functions.

  • We can use albgebraic reasoning to understand programs.

Map

map :: (a -> b) -> [a] -> [b]

> map (+1) [1,3, 4]
> [2,4,5]

Filter

filter :: (a -> Bool) -> [a] -> [a]

> filter even [1,3,4]
> [4]
